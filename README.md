# Green World Multi-Role Platform - Backend

Complete backend API for the Green World distribution, e-commerce, appointment booking, and delivery management platform.

## Features

- 🔐 Firebase Authentication
- 👥 Multi-role user management (Super Admin, Store Owner, Distributor, Customer, Driver)
- 🏪 Multi-store system with subscription management
- 📦 Product and inventory management
- 🛒 Order processing and tracking
- 🚗 Driver delivery management and earnings
- 📅 Appointment booking system
- 📊 Comprehensive reporting and analytics
- 💳 Payment processing integration ready
- 📧 Email notifications
- 🔔 Push notifications (Firebase Cloud Messaging)

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** Firebase Admin SDK
- **File Processing:** ExcelJS, CSV Writer, PDFKit
- **Email:** Nodemailer
- **Scheduling:** Node-cron
